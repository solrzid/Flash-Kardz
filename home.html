About

Flash Kardz is a simple and efficient flashcard web application designed to support effective learning and memory retention. It provides a clean, user-friendly interface that allows users to create, organize, and review flashcards with ease. Ideal for anyone looking to study more effectively, this tool focuses on simplicity and functionality without unnecessary distractions. It was developed out of a desire to make studying more accessible.

This application was developed by Mirasol Pastrana and Hana Zhalei Fajarito, with the goal of creating a focused, easy-to-use resource for learners.
